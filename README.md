3.วิเคราะห์จากผลกระทบจากปัจจัยแวดล้อมต่างๆที่มีผลกระทบต่อจำนวนอุบัติเหตุ ผู้บาดเจ็บหรือเสียชีวิต
ในที่นี้จะพิจารณาถึงปัจัยต่างๆต่อไปนี้ ได้แก่ ประเภทของรถคันที่เกิดอุบัติเหตุ บริเวณที่เกิดเหตุ/ลักษณะทาง 




รูปที่ 6 แสดงความสัมพันธ์ระหว่างประเภทรถคันที่ก่อเหตุ เปรียบเทียบกับจำนวนอุบัติเหตุ จำนวนผู้บาดเจ็บและเสียชีวิต
ตาราง

จากแผนภูมิด้านบนที่แสดงความสัมพันธ์ระหว่างประเภทรถคันที่ก่อเหตุ เปรียบเทียบกับจำนวนอุบัติเหตุ จำนวนผู้บาดเจ็บและเสียชีวิต
จะพบว่ารถปิคอัพอัพบรรทุก 4 ล้อจะมีจำนวนอุบัติเหตุและผู้บาดเจ็บสูงที่สุด ในขณะที่รถจักรยานยนต์จะมีจำนวนผู้เสียชีวิตสูงที่สุดแม้ว่าจำนวนครั้งในการเกิดอุบัติเหตุมีน้อยกว่ารถปิคอัพบรรทุกเกินกว่า 2 เท่า  
ซึสามารถสันนิษฐานได้ว่า อาจจะมีผลมากจากพื้นฐานโครงสร้างทีมีการป้องกันที่น้อยกว่า เช่น การไม่มีโครงสร้างป้องกันแรงกระแทก รวมไปถึงพฤติกรรมการไม่สวมหมวกนิรภัยและการขับขี่ที่มีความเสี่ยงและความเร็วสูง เป็นต้น

ตาราง

จากแผนภูมิด้านบนที่แสดงความสัมพันธ์ระหว่างบริเวณที่เกิดเหตุ เปรียบเทียบกับจำนวนอุบัติเหตุ จำนวนผู้บาดเจ็บ และเสียชีวิต
จะพบว่า บริเวณทางตรงที่ไม่มีความคดเคี้ยว เป็นจุดที่มีจำนวนอุบัติเหตุเกิดขึ้นมากที่สุด โดยมีจำนวนผู้บาดเจ็บและเสียชีวิตสูงเป็นอันดับหนึ่งเช่นกัน 
ซึ่งอาจสันนิษฐานได้ว่า ความเร็วในการขับขี่ที่สูงขึ้นในเส้นทางตรง อาจเพิ่มความเสี่ยงต่อการสูญเสียการควบคุมและเกิดอุบัติเหตุรุนแรง
แม้ว่าทางตรงจะมีทัศนวิสัยที่ดีและไม่มีอุปสรรคจากความคดเคี้ยวของถนน แต่พฤติกรรมการขับขี่ เช่น การขับเร็วเกินกำหนด, การละสายตาจากถนน, หรือการขับขี่โดยประมาท
อาจเป็นปัจจัยสำคัญที่ทำให้เกิดอุบัติเหตุบ่อยครั้ง

4. ข้อสรุปและข้อเสนอแนะสำหรับแนวทางป้องกันลดการเกิดอุบัติเหตุ
จากการศึกษาที่ผ่านมา พบว่าโดยส่วนใหญแล้ว สาเหตุหลักที่ก่อให้เกิดอุบัติเหตุจะมาจากการขับรถที่เกินกว่าอัตราเร็วกำหนด มีจำนวนผู้บาดเจ็บและเสียชีวิตที่สูง
โดยส่วนใหญ่ยานพาหนะคันเกิดเหตุจะเป็นรถจักรยานยนต์ รถปิคอัพบรรทุก 4 ล้อเป็นหลัก และเมื่อพิจารณาจากช่วงเวลาภายใน 1 วัน อุบัติเหตุมักจะเกิดและมีจำนวนผู้บาดเจ้บสูงในช่วงเวลาประมาณ 15-19 นาฬิกา
และในรอบเวลา 1 ปี อุบัติเหตุมักจะเกิดขึ้นในช่วงเดือน มกราคม เมษายนและ ซึ่งจะใก้เคียงช่วงเวลาที่เป็นวันหยุดยาวและมีการสัญจรที่คับคั่งบนถนน

จากการศึกษาที่ผ่านมา พบว่าสาเหตุหลักของอุบัติเหตุบนท้องถนนมักเกิดจากการขับขี่ด้วยความเร็วเกินกว่าที่กฎหมายกำหนด ซึ่งส่งผลให้มีจำนวนผู้บาดเจ็บและเสียชีวิตในระดับสูง โดยเฉพาะในกลุ่ม รถจักรยานยนต์และรถปิคอัพบรรทุก 4 ล้อ ซึ่งเป็นยานพาหนะที่พบว่ามีอัตราการเกิดอุบัติเหตุและมีจำนวนผู้บาดเจ็บหรือเสียชีวิตสูงที่สุด

เมื่อพิจารณาช่วงเวลาภายใน 1 วัน อุบัติเหตุมักเกิดขึ้นบ่อยและมีจำนวนผู้บาดเจ็บสูงสุดในช่วงเวลา 15.00 - 19.00 น. ซึ่งเป็นช่วงที่การจราจรหนาแน่นที่อาจเกิดจากการเดินทางกลับบ้านหลังเลิกงานและเลิกเรียน ทำให้มีโอกาสเกิดอุบัติเหตุจากพฤติกรรมเสี่ยง เช่นความเร่งรีบหรือขับขี่ด้วยความเร็วสูง การเปลี่ยนเลนกะทันหันโดยขาดความระมัดระวัง

และในรอบระยะเวลา 1 ปี อุบัติเหตุมักเกิดขึ้นมากในช่วงเดือนมกราคมและเมษายน ซึ่งสอดคล้องกับช่วงวันหยุดยาว เช่น เทศกาลปีใหม่และสงกรานต์ ที่มีปริมาณการเดินทางเพิ่มขึ้นอย่างมาก ประกอบกับพฤติกรรมเสี่ยง เช่น การขับขี่ระยะทางไกล ความเหนื่อยล้า และการดื่มแอลกอฮอล์ก่อนขับขี่ ซึ่งส่งผลให้จำนวนอุบัติเหตุและความรุนแรงของการบาดเจ็บเพิ่มขึ้นในช่วงเวลาดังกล่าวอีกด้วย

ซึ่งเป็นสิ่งที่สามารถป้องกันหรือปรับปรุงได้โดยช้เทคโนโลยีหรือแนวทางอื่นๆเข้ามาประยุกต์ใช้

ตัวอย่างแนวทางป้องกันและลดอุบัติเหตุ
มาตรการควบคุมและบังคับใช้กฎหมาย การช่วยเหลือเมื่อเกิดอุบัติเหตุ
+ เพิ่มจุดตรวจตราและด่านสกัด บนถนนสายหลักและจุดที่มีสถิติอุบัติเหตุสูงหรือในเวลาและบริเวณที่มักมีอุบัติเหตุเกิดขึ้นบ่อยๆเพื่อเฝ้าระวังพฤติกรรมการขับขี่ที่เสี่ยงต่อการเกิดอุบัติเหตุ
+ เพิ่มจุดช่วยเหลือพยาบาลฉุกเฉิน ในบริเวณและช่วงเวลาที่เกิดอุบัติเหตุถี่ เพื่อให้สามารถให้การช่วยเหลือผู้บาดเจ็บได้อย่างรวดเร็ว ลดความสูญเสียที่อาจเกิดขึ้น

การพัฒนาระบบช่วยเหลือการขับขี่เพื่อป้องกันอุบัติเหตุ
+ การขับรถเร็วเกินกำหนด:ติดตั้ง ระบบควบคุมและจำกัดความเร็วอัจฉริยะ (Intelligent Speed Assistance - ISA) ที่สามารถทำงานสัมพันธ์กับ ป้ายสัญญาณจราจรและข้อมูลจำกัดความเร็วของแต่ละพื้นที่ เพื่อช่วยให้สามารถรักษาความเร็วให้อยู่ในช่วงที่ปลอดภัยกับบริเวณที่ขับขี่
+ เพิ่มกล้องตรวจจับความเร็วและระบบแจ้งเตือนอัตโนมัติ เพื่อกวดขันให้ผู้ใช้ท้องปฏิบัติตามกฎระเบียบจราจร

การดื่มสุราและหลับใน:
+ ใช้ระบบตรวจจับระดับแอลกอฮอล์อัตโนมัติในรถยนต์ (Alcohol Detection System) ซึ่งสามารถปิดกั้นการสตาร์ตรถ หากพบว่าผู้ขับขี่มีระดับแอลกอฮอล์เกินเกณฑ์ที่กำหนด
+ ใช้ระบบตรวจจับอาการเหนื่อยล้าและขาดสมาธิของผู้ขับขี่ (Driver Monitoring System - DMS) ที่สามารถเตือนผู้ขับขี่ให้กลับมามีสมาธิ หรือแจ้งเตือนให้หยุดพักเมื่อมีอาการง่วงนอนหรืออ่อนล้า
